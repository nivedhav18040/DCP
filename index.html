How it works

Discover applications on endpoints, build control policies, deploy them through agent-server sync, and enforce execution with continuous monitoring.

1

Discover applications

After agent installation, a one-time scan collects verified executables (Program Files / Program Files (x86)) and running processes so your console gets a clear software inventory.
